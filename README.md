# Descriptive Sales Analysis with Pandas

## 📊 Project Overview
This project performs a descriptive analysis of a sales dataset using Python and Pandas.

## 🛠️ Technologies
- Python
- Pandas
- Google Colab

## 🔄 Data Transformation
- Combined date and time into datetime
- Removed unnecessary columns
- Created sales_amount metric
- Standardized gender column
- Renamed columns to lowercase

## 📈 Analysis
- Sales metrics (max, min, mean, median)
- Product categories
- Top 5 products
- Customer behavior by gender
- Payment methods

## 💡 Insights
- Sales are concentrated in a few categories
- Some products dominate demand
- Gender differences in purchase behavior
- One payment method stands out
